# 7.6 Stringovi

[Sadržaj](_00.0-sr.md)

Na vebu, skoro sve što vidimo (uključujući korisničke unose, pristup bazi podataka itd.), predstavljeno je stringovima. Oni su veoma važan deo veb razvoja. U mnogim slučajevima, takođe nam je potrebno da delimo, spajamo, konvertujemo i na drugi način manipulišemo stringovima. U ovom odeljku ćemo predstaviti pakete `strings` i `strconv` iz standardne Go biblioteke.

## Paket strings

Sledeće funkcije su iz `strings` paketa. Više detalja potražite u zvaničnoj dokumentaciji:

- **Contains**

  ```go
  func Contains(s, substr string) bool
  ```
  
  Proverava da li string `s` sadrži string `substr`, vraća bulovsku vrednost.
  
  ```go
  fmt.Println(strings.Contains("seafood", "foo"))
  fmt.Println(strings.Contains("seafood", "bar"))
  fmt.Println(strings.Contains("seafood", ""))
  fmt.Println(strings.Contains("", ""))
  ```
  
  Output:
  
  ```sh
  true
  false
  true
  true
  ```

- **Join**

  ```go
  func Join(a []string, sep string) string
  ```
  
  Kombinuje stringove iz isečka `a` pomoću separatora `sep`.
  
  ```go
  s := []string{"foo", "bar", "baz"}
  fmt.Println(strings.Join(s, ", "))
  //Output:foo, bar, baz
  ```

- **Index**

  ```go
  func Index(s, sep string) int
  ```

  Pronalazi indeks `sep` u stringu `s`, vraća `-1` ako nije pronađen.
  
  ```go
  fmt.Println(strings.Index("chicken", "ken"))
  fmt.Println(strings.Index("chicken", "dmr"))
  //Output:4
  //-1
  ```

- Repeat

  ```go
  func Repeat(s string, count int) string
  ```
  
  Ponovite string s count puta.
  
  ```go
  fmt.Println("ba" + strings.Repeat("na", 2))
  //Output:banana
  ```

- **Replace**

  ```go
  func Replace(s, old, new string, n int) string
  ```
  
  Zameni string `old` sa stringom `new` u stringu `s`. `n` je broj zamena. Ako je `n` manje od 0, zameni sve instance.
  
  ```go
  fmt.Println(strings.Replace("oink oink oink", "k", "ky", 2))
  fmt.Println(strings.Replace("oink oink oink", "oink", "moo", -1))
  //Output:oinky oinky oink
  //moo moo moo
  ```

- **Split**

  ```go
  funkcija Split(s, sep string) []string
  ```
  
  Podeli string `s` pomoću separatora `sep` na isečak.
  
  ```go
  fmt.Printf("%q\n", strings.Split("a,b,c", ","))
  fmt.Printf("%q\n", strings.Split("a man a plan a canal panama", "a "))
  fmt.Printf("%q\n", strings.Split(" xyz ", ""))
  fmt.Printf("%q\n", strings.Split("", "Bernardo O'Higgins"))
  //Output:["a" "b" "c"]
  //["" "man " "plan " "canal panama"]
  //[" " "x" "y" "z" " "]
  //[""]
  ```

- **Trim**

  ```go
  func Trim(s string, cutset string) string
  ```
  
  Uklanja `cutset` string iz stringa `s` ako je krajnje levo ili krajnje desno.
  
  ```go
  fmt.Printf("[%q]", strings.Trim(" !!! Achtung !!! ", "! "))
  // Output:["Achtung"]
  ```

- **Field**

```go
func Fields(s string) []string
```

Uklanja elemente razmaka i delite string sa razmakom na isečkom.

```go
fmt.Printf("Fields are: %q", strings.Fields("  foo bar  baz   "))
//Output:Fields are: ["foo" "bar" "baz"]
```

## Paket strconv

Sledeće funkcije su iz strconv paketa. Kao i obično, pogledajte zvaničnu dokumentaciju za više detalja:

- **Append***

  Dodaj seriju, konvertuj podatke u string i dodaj trenutnom bajtnom isečku.
  
  ```go
  package main
  
  import (
      "fmt"
      "strconv"
  )
  
  func main() {
      str := make([]byte, 0, 100)
      str = strconv.AppendInt(str, 4567, 10)
      str = strconv.AppendBool(str, false)
      str = strconv.AppendQuote(str, "abcdefg")
      str = strconv.AppendQuoteRune(str, '单')
      fmt.Println(string(str))
  }
  ```

- **Format***  

  Formatiranje serija, konvertovanje drugih tipova podataka u string.
  
  ```go
  package main
  
  import (
      "fmt"
      "strconv"
  )
  
  func main() {
      a := strconv.FormatBool(false)
      b := strconv.FormatFloat(123.23, 'g', 12, 64)
      c := strconv.FormatInt(1234, 10)
      d := strconv.FormatUint(12345, 10)
      e := strconv.Itoa(1023)
      fmt.Println(a, b, c, d, e)
  }
  ```

- **Parse***

  Raščlanjivanje serija, konvertovanje stringova u druge tipove.
  
  ```go
  package main
  
  import (
      "fmt"
      "strconv"
  )
  
  func main() {
      a, err := strconv.ParseBool("false")
      if err != nil {
          fmt.Println(err)
      }
      b, err := strconv.ParseFloat("123.23", 64)
      if err != nil {
          fmt.Println(err)
      }
      c, err := strconv.ParseInt("1234", 10, 64)
      if err != nil {
          fmt.Println(err)
      }
      d, err := strconv.ParseUint("12345", 10, 64)
      if err != nil {
          fmt.Println(err)
      }
      e := strconv.Itoa(1023)
      fmt.Println(a, b, c, d, e)
  }
  ```
