
# 10.3 Međunarodne lokacije

[Sadržaj](_00.0-sr.md)

U prethodnom odeljku je objašnjeno kako se obrađuju lokalizovani resursi, naime korišćenjem lokalnih konfiguracionih datoteka. Šta možemo da uradimo ako treba da se bavimo više lokalizovanih resursa kao što su prevodi teksta, vremena i datumi, brojevi itd.? Ovaj odeljak će se baviti ovim problemima jedno po jedno.

## Upravljanje sa više lokalnih paketa

Prilikom razvoja aplikacije, često je prva stvar koju treba da uradite da odlučite da li želite da podržite više od jednog jezika. Ako odlučite da podržite više jezika, moraćete da razvijete organizacionu strukturu kako biste olakšali proces dodavanja više jezika u budućnosti. Jedan od načina na koji to možemo da uradimo je da sve naše povezane datoteke lokalnih podešavanja stavimo zajedno u direktorijum "config/locales" ili nešto slično. Pretpostavimo da želite da podržite i kineski i engleski jezik. U ovom slučaju, stavili biste i datoteke lokalnih podešavanja, "en.json" i "zh.json", u gore pomenutu fasciklu. Njihov sadržaj bi verovatno izgledao otprilike ovako:

"zh.json"

```json
{
"zh": {
    "submit": "提交",
    "create": "创建"
    }
}
```

"en.json"

```json
{
"en": {
    "submit": "Submit",
    "create": "Create"
    }
}
```

Odlučili smo da koristimo neke Go pakete trećih strana kako bismo lakše internacionalizovali naše veb aplikacije. U slučaju [go-i18n](https://github.com/astaxie/go-i18n) (napredniji i18n paket možete pronaći [ovde](https://github.com/beego/i18n)), prvo moramo da registrujemo naš "config/locales" direktorijum da bismo učitali sve naše lokalne datoteke:

```go
Tr := i18n.NewLocale()
Tr.LoadPath("config/locales")
```

Ovaj paket je jednostavan za korišćenje. Možemo testirati da li radi ovako:

```go
fmt.Println(Tr.Translate("submit"))
//Output "submit"

Tr.SetLocale("zn")
fmt.Println(Tr.Translate("submit"))
//Outputs "递交"
```

## Automatski učitaj lokalni paket

Upravo smo opisali kako se automatski učitavaju prilagođeni jezički paketi. U stvari, `go-i18n` biblioteka dolazi unapred učitana sa gomilom podrazumevanih informacija o formatiranju, kao što su formati vremena i valute. Korisnici mogu da zamene i prilagode ove podrazumevane konfiguracije svojim potrebama. Razmotrite sledeći postupak:

```go
//Load the default configuration files, which are placed below in `go-i18n/locales`

//File should be named zh.json, en-json, en-US.json etc., so we can continuously support more languages

func (il *IL) loadDefaultTranslations(dirPath string) error {
    dir, err := os.Open(dirPath)
    if err != nil {
        return err
    }
    defer dir.Close()

    names, err := dir.Readdirnames(-1)
    if err != nil {
        return err
    }

    for _, name := range names {
        fullPath := path.Join(dirPath, name)

        fi, err := os.Stat(fullPath)
        if err != nil {
            return err
        }

        if fi.IsDir() {
            if err := il.loadTranslations(fullPath); err != nil {
                return err
            }
        } else if locale := il.matchingLocaleFromFileName(name); locale != "" {
            file, err := os.Open(fullPath)
            if err != nil {
                return err
            }
            defer file.Close()

            if err := il.loadTranslation(file, locale); err != nil {
                return err
            }
        }
    }

    return nil
}
```

Koristeći gornji kod za učitavanje svih naših podrazumevanih prevoda, možemo zatim koristiti sledeći kod za izbor i korišćenje lokaliteta:

```go
fmt.Println(Tr.Time(time.Now()))
//Output: 2009年1月08日 星期四 20:37:58 CST

fmt.Println(Tr.Time(time.Now(),"long"))
//Output: 2009年1月08日

fmt.Println(Tr.Money(11.11))
//Output: ¥11.11
```

## Datoteke šablona

Iznad smo predstavili jedan način upravljanja i integracije brojnih jezičkih paketa. Neke od funkcija koje smo implementirali zasnovane su na logičkom sloju, na primer: „Tr.Translate“, „Tr.Time“, „Tr.Money“ i tako dalje. U logičkom sloju možemo koristiti ove funkcije (nakon što navedemo potrebne parametre) za primenu vaših prevoda, prikazujući rezultate direktno u sloju šablona tokom renderovanja. Šta možemo da uradimo ako želimo da koristimo ove funkcije direktno u sloju šablona? U slučaju da ste zaboravili, ranije u knjizi smo pomenuli da Go šabloni podržavaju prilagođene funkcije šablona. Sledeći kod pokazuje koliko je lako implementirati `map` func:

### Tekst

Jednostavna funkcija za konverziju teksta koja implementira mapfunc može se videti ispod. Ona koristi `Tr.Translate` za izvršavanje odgovarajućih prevoda:

```go
func I18nT(args ...interface{}) string {
    ok := false
    var s string
    if len(args) == 1 {
        s, ok = args[0].(string)
    }
    if !ok {
        s = fmt.Sprint(args...)
    }
    return Tr.Translate(s)
}
Funkciju registrujemo ovako:

t.Funcs(template.FuncMap{"T": I18nT})
Zatim ga koristite iz svog šablona:

{{.V.Submit | T}}
```

### Datum i vreme

Datumi i vremena pozivaju `Tr.Time` funkciju da izvrši njihove prevode. Funkcija map je implementirana na sledeći način:

```go
func I18nTimeDate(args ...interface{}) string {
    ok := false
    var s string
    if len(args) == 1 {
        s, ok = args[0].(string)
    }
    if !ok {
        s = fmt.Sprint(args...)
    }
    return Tr.Time(s)
}
```

Registrujte funkciju na sledeći način:

```go
t.Funcs(template.FuncMap{"TD": I18nTimeDate})
Zatim ga koristite iz svog šablona:

{{.V.Now | TD}}
```

### Valuta

Valute koriste Tr.Moneyfunkciju za konvertovanje novca. Funkcija mapFunc je implementirana na sledeći način:

```go
func I18nMoney(args ...interface{}) string {
    ok := false
    var s string
    if len(args) == 1 {
        s, ok = args[0].(string)
    }
    if !ok {
        s = fmt.Sprint(args...)
    }
    return Tr.Money(s)
}
```

Registrujte funkciju na sledeći način:

```go
t.Funcs(template.FuncMap{"M": I18nMoney})
```

Zatim ga koristite iz svog šablona:

```html
{{.V.Money | M}}
```

## Rezime

U ovom odeljku smo naučili kako da implementiramo više jezičkih paketa u našim veb aplikacijama. Videli smo da pomoću prilagođenih jezičkih paketa ne samo da možemo lako internacionalizovati naše aplikacije, već i olakšati dodavanje drugih jezika (korišćenjem konfiguracione datoteke). Podrazumevano, paket go-i18n će pružiti neke uobičajene konfiguracije za vreme, valutu itd., što može biti veoma pogodno za korišćenje. Naučili smo da se ove funkcije mogu koristiti i direktno iz naših šablona pomoću funkcija mapiranja; svaki prevedeni string može se direktno proslediti našim šablonima. Ovo omogućava našim veb aplikacijama da prihvate više jezika uz minimalan napor.
