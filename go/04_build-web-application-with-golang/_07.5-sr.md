# 7.5 Datoteke

[Sadržaj](_00.0-sr.md)

Datoteke su neophodni objekti na svakom računarskom uređaju. Neće vas iznenaditi da ih i veb aplikacije intenzivno koriste. U ovom odeljku ćemo naučiti kako da radimo sa datotekama u programskom jeziku Go.

## Direktorijumi

U programu Go, većina funkcija za rad sa datotekama nalazi se u `os` paketu. Evo nekih funkcija direktorijuma:

- **Mkdir**

  ```go
  func Mkdir(name string, perm FileMode) error
  ```
  
  Kreira direktorijum sa `name`. `Perm` su dozvole direktorijuma, tj. 0777.

- **MkdirAll**

  ```go
  func MkdirAll(path string, perm FileMode) error
  ```
  
  Kreira više direktorijuma prema `path`, kao što je "astaxie/test1/test2".

- Remove

  ```go
  func Remove(name string) error
  ```
  
  Uklanja direktorijum sa `name`. Vraća grešku ako nije direktorijum ili nije prazan.

- **RemoveAll**  

  ```go
  func RemoveAll(path string) error
  ```

Uklanja više direktorijuma u ​​skladu sa `path`. Direktorijumi neće biti obrisani ako `path` je jedna putanja.

Primer koda:

```go
package main

import (
    "fmt"
    "os"
)

func main() {
    os.Mkdir("astaxie", 0777)
    os.MkdirAll("astaxie/test1/test2", 0777)
    err := os.Remove("astaxie")
    if err != nil {
        fmt.Println(err)
    }
    os.RemoveAll("astaxie")
}
```

## Datoteke

### Kreiranje i otvaranje datoteke

Postoje dve funkcije za kreiranje datoteka:

- **Create

  ```go
  func Create(name string) (file *File, err Error)
  ```
  
  Kreira datoteku imena `name` i vraća objekat `*File` datoteke koji se može čitati i pisati sa dozvolom 0666.

- **NewFile**

  ```go
  func NewFile(fd uintptr, name string) *File
  ```

  Kreira novu datoteku koja već postoji, ima hendl `fd` i vraća objekat datoteke `*File`.

Takođe postoje dve funkcije za otvaranje datoteka:

- **Open**

  ```go
  func Open(name string) (file *File, err Error)
  ```

  Otvara datoteku `name` samo za čitanje, pozivajući `OpenFile` ispod haube. Vraća objekat `*File`.

- **OpenFile**

  ```go
  func OpenFile(name string, flag int, perm uint32) (file *File, err Error)
  ```

  Otvara datoteku pod nazivom `name`. `flag` je režim otvaranja kao što je samo za čitanje, čitanje-pisanje itd. `Perm` su dozvole za datoteku. Vraća objekat datoteke `*File`.

### Pisanje datoteka

Funkcije za pisanje datoteka:

- **Write**

  ```go
  func (file *File) Write(b []byte) (n int, err Error)
  ```

  Upisuje sadržaj `b` tipa isečka bajtova u datoteku `*File`.

- **WriteAt**

  ```go
  func (file *File) WriteAt(b []byte, off int64) (n int, err Error)
  ```

  Upisuje sadržaj isečka bajtova `b` na određenu poziciju `off` u datoteci `*File`.

- **WriteString**

  ```go
  func (file *File) WriteString(s string) (ret int, err Error)
  ```

  Upisuje string `s` u datoteku `*File`.

Primer koda:

```go
package main

import (
    "fmt"
    "os"
)

func main() {
    userFile := "astaxie.txt"
    fout, err := os.Create(userFile)        
    if err != nil {
        fmt.Println(userFile, err)
        return
    }
    defer fout.Close()
    for i := 0; i < 10; i++ {
        fout.WriteString("Just a test!\r\n")
        fout.Write([]byte("Just a test!\r\n"))
    }
}
```

### Čitanje datoteka

Funkcije za čitanje datoteka:

- **Read**

  ```go
  func (file *File) Read(b []byte) (n int, err Error)
  ```
  
  Čita podatke iz `*File` u `b` isečak bajtova.

- **ReadAt**

  ```go
  func (file *File) ReadAt(b []byte, off int64) (n int, err Error)
  ```

  Čita podatke iz `*File` od pozicije `off` u isečak `b`. Vraća broj pročitanih bajtova.

Primer koda:

```go
package main

import (
    "fmt"
    "os"
)

func main() {
    userFile := "asatxie.txt"
    fl, err := os.Open(userFile)        
    if err != nil {
        fmt.Println(userFile, err)
        return
    }
    defer fl.Close()
    buf := make([]byte, 1024)
    for {
        n, _ := fl.Read(buf)
        if 0 == n {
            break
        }
        os.Stdout.Write(buf[:n])
    }
}
```

### Brisanje datoteke

Go koristi istu funkciju za brisanje datoteka i direktorijuma:

- **Remove

  ```go
  func Remove(name string) Error
  ```

  Uklonja datoteku ili direktorijum pod nazivom name.

> [!Note]
> `name/` označava da je u pitanju direktorijum.
