
# 7.7 Rezime

[Sadržaj](_00.0-sr.md)

U ovom poglavlju smo predstavili neke alate za obradu teksta kao što su XML, JSON, Regexp, a takođe smo govorili i o šablonima. XML i JSON su alati za razmenu podataka. Možete predstaviti skoro svaku vrstu informacija koristeći ova dva formata. Regexp je moćan alat za pretragu, zamenu i isecanje tekstualnog sadržaja. Pomoću šablona možete lako kombinovati dinamičke podatke sa statičkim datotekama. Svi ovi alati su korisni pri razvoju veb aplikacija. Nadam se da sada bolje razumete obradu i prikazivanje sadržaja pomoću Go-a.
